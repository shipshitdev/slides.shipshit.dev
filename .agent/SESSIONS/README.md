# Sessions

**Purpose:** Daily session documentation for AI continuity.
**Last Updated:** 2025-12-29

---

## ONE FILE PER DAY

**Correct:** `2025-01-15.md`
**Wrong:** `2025-01-15-feature-name.md`

Multiple sessions same day = Session 1, Session 2, etc. in the SAME file.

---

## File Format

See `TEMPLATE.md` for the session file template.

---

## Quick Reference

### At Session Start

1. Check if today's file exists
2. If yes: read it for context
3. If no: create using template

### During Session

Track:
- Decisions made
- Files changed
- Patterns used
- Mistakes and fixes

### At Session End

Add session entry with:
- What was done
- Files changed
- Decisions with rationale
- Next steps
